代表メッセージ

私ども、株式会社グランドプランはお客様のお役に立てる売買に重点をおいて、お付き合いをさせていただいております。 一言で不動産売買と言っても、お客様の状況は千差万別、ケースバイケースであり、常に同じお取り引きができる業種ではございません。 そのため、多角的な面から様々な状況を綿密に検討し、お客様に最善のご提案を行えるようにとスタッフ一同、日々情報収集や状況の変化を敏感に対応できるよう努力致しております。 売却をご希望されるお客様であれば様々なご事情がお有りのことと思います。 その時、安心して私どもをご指名いただけるように、調査・相談・スピーディーな売却活動・そしてご購入者の決定・契約・アフターサービスに至るまで、適確でスムーズなお取り引きを心掛けております。また、急々な場合や、状況に応じて物件の買取も致しております。 「売れずに困っている」「早く株式会社グランドプランと知り合えば良かった」というお声をよくいただきますが、これからも「株式会社グランドプランに頼んで良かった」というお声をいただけるようお客様主義で邁進致します。目先の利益だけを追求するのではなく、お客様とこれからも永く良い関係が結べるように、誠心誠意お付き合いさせていただきます。 どうぞ、株式会社グランドプランを宜しくお願い申し上げます。 代表取締役 衛藤 啓二
